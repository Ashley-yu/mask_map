## Demo
https://ashley-yu.github.io/mask_map/

## 簡介
台灣於 2020/02/06 實施口罩購買實名制，為方便民眾查詢各藥局口罩庫存量，供民間使用開放資料製作各種查詢載體。

本網頁提供功能如下：
* 定位搜尋附近藥局
* 輸入縣市或地區搜尋藥局
* 可篩選尚有成人或兒童口罩庫存藥局
* 查詢結果依顏色區分庫存
  * 成人：綠色
  * 兒童：粉色
  * 無庫存：灰色
* 提供藥局導航連結
* 依星期提示可購買族群
